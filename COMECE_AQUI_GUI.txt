  ✅ Ícone azul na bandeja

───────────────
    copy NUL "%USERPROFILE%\Downloads\teste.pdf"


Passo 5: Ver acontecer ✨

  ✅ Notificação aparece na tela

🎨 TOUR RÁPIDO DA GUI
════════════════════════════════════════════════════════════════

A JANELA TEM 3 ABAS:

1️⃣ 📊 DASHBOARD (Aba padrão)
   ├── Informações de pastas
   ├── Data/hora de última atualização
   ├── Contadores coloridos (0, 1, 2...)
   └── Botões de ação rápida

2️⃣ 📋 LOG (Histórico)
   ├── Lista completa de arquivos movidos
   ├── Com timestamps
   ├── Botão para limpar
   └── Botão para exportar em .txt

3️⃣ ⚙️ CONFIGURAÇÕES
   ├── Mostrar caminhos das pastas
   ├── Abrir direto no Explorer
   ├── Lista de todas as categorias
   └── Informações do projeto


════════════════════════════════════════════════════════════════
🎮 CONTROLES PRINCIPAIS
════════════════════════════════════════════════════════════════

JANELA:
┌─────────────┐
│ [_] [-] [X] │  Minimizar, Maximizar, Fechar
└─────────────┘

BOTÕES:
┌────────────────────────────────────────┐
│ 🔄 Atualizar   → Força refresh         │
│ 📁 Abrir       → Abre pasta final      │
│ ➖ Minimizar   → Oculta janela         │
│ 🛑 Sair        → Encerra programa     │
└────────────────────────────────────────┘

TRAY (Botão Direito):
┌────────────────────────────────────────┐
│ 🪟 Abrir Janela   → Mostra GUI         │
│ 📁 Organizador    → Abre pasta final   │
│ 📂 Downloads      → Abre Downloads     │
│ 🛑 Sair           → Fecha tudo        │
└────────────────────────────────────────┘


════════════════════════════════════════════════════════════════
💻 REQUISITOS
════════════════════════════════════════════════════════════════

MÍNIMO:
  • Python 3.7+
  • Windows, macOS ou Linux
  • RAM: 4GB (recomendado)
  • Disco: 200MB livre

DEPENDÊNCIAS:
  • PyQt5 (GUI)
  • pystray (tray icon)
  • pillow (ícone)
  • watchdog (monitor)
  • plyer (notificações)

INSTALAR TUDO:
  pip install -r requirements.txt
