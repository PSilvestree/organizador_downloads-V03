# 🚀 Quick Start - 5 Minutos

## 1️⃣ Instalar (30 segundos)

```bash
pip install -r requirements.txt
```

## 2️⃣ Executar (2 segundos)

```bash
python organizador_downloads_com_gui.py
```

## 3️⃣ Testar (1 minuto)

Coloque um arquivo em Downloads:

**Windows (PowerShell):**
```bash
copy NUL "$env:USERPROFILE\Downloads\teste.pdf"
```

**macOS/Linux:**
```bash
touch ~/Downloads/teste.txt
```

## 4️⃣ Ver acontecer (30 segundos)

- ✅ Dashboard mostra 1 arquivo
- ✅ Log mostra onde foi movido
- ✅ Arquivo está em `~/Organizador/PDFs/`
- ✅ Notificação apareceu

## 🎉 Pronto!

Seu organizador está funcionando!
